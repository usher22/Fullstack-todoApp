Fullstack Todo App
A modern, full-stack Todo application built using Next.js, Tailwind CSS, Express.js, Sequelize, and PostgreSQL.  
This project demonstrates a clean, production-ready architecture for handling full CRUD operations with a RESTful backend and a responsive frontend.

Tech Stack

Frontend
- [Next.js](https://nextjs.org/) — React framework for server-side rendering and routing  
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first CSS framework for modern, responsive design  
- [Axios](https://axios-http.com/) — For making API requests to the backend  

Backend
- [Node.js](https://nodejs.org/) — JavaScript runtime for the server  
- [Express.js](https://expressjs.com/) — Lightweight web framework for building REST APIs  
- [Sequelize](https://sequelize.org/) — ORM for interacting with PostgreSQL  
- [PostgreSQL](https://www.postgresql.org/) — Relational database for persistent data storage  

Features
- Create, Read, Update, and Delete (CRUD) todos  
- Responsive UI built with Tailwind CSS  
- Secure API endpoints  
- Fast performance with optimized Next.js pages  
- Seamless frontend-backend integration via Axios  
- Deployed frontend (Vercel) and backend (Node.js server)  
- Environment variable configuration for easy setup
